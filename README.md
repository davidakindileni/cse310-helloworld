# Overview

As a software engineer, I continually improve my development workflow and programming skills by building small foundational projects that deepen my understanding of software development tools and practices. This project was created to verify that my local Python development environment, source control workflow, and GitHub integration are functioning correctly.

This software is a simple “Hello World!” application written in Python 3.14.5. The program displays a greeting message in the terminal to confirm that Python is properly installed and configured within the development environment.

The purpose of writing this software was to test and validate the complete development setup using Visual Studio Code, Python, Git, and GitHub. It also provided an opportunity to practice creating repositories, committing code changes, and successfully pushing code to the GitHub repository named `cse310-helloworld`.

[Software Demo Video](https://youtu.be/EdVYPNdwOJ4)

# Development Environment

The following tools were used to develop this software:

- Visual Studio Code (VS Code)
- Python 3.14.5
- Git
- GitHub

The software was written using the Python programming language. No external libraries were required for this project since the application only uses Python’s built-in `print()` function.
